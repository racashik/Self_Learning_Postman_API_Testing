We can use "Dynamic random variable" which generate random value.
Dynamic inbuild variable starts with "$" sign i.e: {{$randomFirstName}}

Below has a sample of use of Dynamic Variable

{
    "name": "{{$randomFirstName}}",
    "job": "{{$randomJobTitle}}"
}
